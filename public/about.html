About This Project

This University Management System is a Node.js-based mini project for MCA students, built to demonstrate full-stack development skills.

Features

  • Manage student records including adding, editing, and deleting students.
  • Interactive and responsive user interface for seamless navigation.
  • Preloader for enhanced user experience during page load.

Technologies Used

  • Frontend: HTML, CSS, JavaScript
  • Backend: Node.js, Express.js
  • Database: MongoDB

Purpose

The primary goal of this project is to provide a practical demonstration of full-stack development, showcasing the integration of frontend and backend technologies to build a functional web application.

About the Developers

This project was developed by Ayush Thakur and Ritish Chauhan, students of Chandigarh University, pursuing their Master of Computer Applications (MCA). They are passionate about software development and have created this project as part of their academic curriculum to enhance their skills in full-stack development.
